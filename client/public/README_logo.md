# ブランドロゴの配置

LPのヘッダー・ヒーロー・フッター、およびSEO用の構造化データで使用する**会社（ブランド）ロゴ**を反映するには、このフォルダ（`client/public`）に次のファイルを配置してください。

- **ファイル名**: `logo.png`
- **配置場所**: `client/public/logo.png`

配置すると、管理画面で別のロゴを設定していない限り、この画像がLP全体のロゴとして表示されます。未配置の場合はデフォルトのアイコン（SVG）が表示されます。

※ 管理画面（ImageManager / ContentsManager）でアップロードしたロゴは、ローカルの `logo.png` より優先されます。
